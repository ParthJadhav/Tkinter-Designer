## 💡 प्रस्तावना

Tkinter डिजाइनर का उपयोग Python में GUI (ग्राफिकल यूजर इंटरफेस) विकास प्रक्रिया को गति देने के लिए  बनाया गया था। यह Python में सुंदर Tkinter GUI बनाने के लिए प्रसिद्ध डिजाइन सॉफ्टवेयर [Figma](https://www.figma.com/) का उपयोग करता है।

Tkinter डिज़ाइनर एक डिज़ाइन फ़ाइल का विश्लेषण करने और GUI के लिए आवश्यक संबंधित code और फ़ाइलें बनाने के लिए Figma API का उपयोग करता है। यहां तक ​​​​कि Tkinter डिजाइनर का GUI भी Tkinter डिजाइनर का उपयोग करके बनाया गया है।

<img width="500" alt="Tkinter Designer GUI" src="https://user-images.githubusercontent.com/42001064/119863796-92af4a80-bf37-11eb-9f6c-61b1ab99b039.png">

## ☄️ Tkinter डिजाइनर के लाभ

1. ड्रैग एंड ड्रॉप इंटरफेस (खींचकर छोड़ने वाला इंटरफेस)
2. मैन्युअल रूप से कोड बनाने की तुलना में महत्वपूर्ण रूप से तेज़ है
3. अधिक सुंदर इंटरफेस बनाने की क्षमता

## 📐 यह काम किस प्रकार करता है

केवल एक चीज जो उपयोगकर्ता को करने की आवश्यकता है वह है Figma के साथ एक इंटरफ़ेस डिज़ाइन करना, और फिर Figma फ़ाइल URL और API (एप्लिकेशन प्रोग्रामिंग इंटरफेस) टोकन को Tkinter डिजाइनर में पेस्ट करना है।

Tkinter डिज़ाइनर स्वचालित रूप से Tkinter में GUI (ग्राफिकल यूजर इंटरफेस) बनाने के लिए आवश्यक सभी कोड और छवियों(images) को उत्पन्न करेगा।
